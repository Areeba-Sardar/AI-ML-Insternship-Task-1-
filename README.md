# AI/ML Internship Task

This repository contains AI/ML internship projects focused on machine learning, data analysis, predictive modeling, and chatbot development. Each task demonstrates practical implementation of AI and machine learning concepts using Python and popular data science libraries.

## Tasks Overview

### Task 1 – Iris Dataset Exploration & Visualization
Performed data loading, inspection, statistical analysis, and visualization of the Iris dataset using Pandas, Matplotlib, and Seaborn.

### Task 2 – Stock Price Prediction
Built a regression model to predict future stock closing prices using historical stock market data obtained through Yahoo Finance.

### Task 3 – Heart Disease Prediction
Developed a classification model to predict the likelihood of heart disease based on patient health attributes and medical data.

### Task 4 – General Health Query Chatbot
Created a health-related chatbot capable of answering general medical questions while following basic safety guidelines.

### Task 5 – Mental Health Support Chatbot
Built a supportive chatbot trained to provide empathetic responses for stress, anxiety, and emotional wellness conversations.

### Task 6 – House Price Prediction
Implemented a regression model to estimate house prices using property features such as area, bedrooms, and location.
